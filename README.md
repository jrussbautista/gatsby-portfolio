## Installation

- Clone repository
  <br>
  ```
  git clone https://github.com/jrussumbrella/gatsby-portfolio
  ```
- Navigate into directory and install dependencies
  ```
  cd gatsby-portfolio
  npm install
  ```
- Start it up
  ```
   gatsby develop
  ```
